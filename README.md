C51单片机小项目，超声波测距显示软件，实时显示超声波模块通过蓝牙传来的距离数据，

蓝牙是经典蓝牙，蓝牙模块用的是HC-06，

C51接线和代码请看这里 https://github.com/bunny-chz/BTSerial_C51

蓝牙封装库引用的是 HBluetooth: https://github.com/g-HJY/HBluetooth
